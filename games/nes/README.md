# Thirty Seven — NES Style

A Parable of Desire, Death, and the Undying Cycle.

2D Canvas rendition with NES-inspired aesthetics: 256×240 resolution, limited palette, square-wave chiptune audio.

## Run

Open `index.html` or serve:

```bash
cd games/nes
python -m http.server 8000
# Open http://localhost:8000
```

## Controls

- **D-pad / Left stick / WASD**: Move the Wanderer
- **A / B / X / Y / Start / Z / X / Enter**: BREATHE (first tap), LIGHTS OUT (second tap during breath)
- **Enter / Space / A / Start**: Start game, Continue

## Features

- Same mechanics as web 2D: awaken souls, avoid Pale Ones, breathe → lights out → rise
- Constellation layouts, 37-second timer, 3–7 second breathing, progressive awakening (level 10+)
- NES color palette, pixel scaling, Press Start 2P font

*The fire never dies.*
