# Thirty Seven — A Pygame Parable

A video game adaptation of the screenplay *Thirty Seven: A Parable of Desire, Death, and the Undying Cycle*.

## How to Run

```bash
cd games/python
pip install -r requirements.txt
python main.py
```

## How to Play

- **WASD** or **Arrow keys**: Move the Wanderer
- **SPACE**: First press begins *Breathe* (a brief pause before the fall)
- **SPACE** (again during Breathe): *Lights Out* — voluntary death. You fall, then rise again, briefly invulnerable.
- **Goal**: Awaken the crowd (get near them and stay near—they fill with light) while avoiding the Pale Ones.
- **Win**: Awaken enough souls in each era to progress. Complete all six levels.

## The Cycle

- **While true. Do.** — You are the Wanderer. You carry the song.
- **Breathe** — A moment of yielding before the fall. Use it to prepare.
- **Fall** — Voluntary death. The Pale Ones cannot catch you when you're falling or rising.
- **Rise** — You return. Invulnerable for a moment. The cycle continues.
- **Done** — Until the next iteration.

## Eras

1. **The Birthplace** — Learn the cycle.
2. **Old World Tavern** — Candlelit. The Pale Ones appear.
3. **Neon Alley** — Future city. More hunters.
4. **Forest** — Ancient. The crowd grows.
5. **The Crowd** — Shoulder to shoulder. Many to awaken.
6. **Lights Out** — Darkness. The final trial.
